# Agent Notes

This repository is an awesome list for open source AI-era developer tools.

## Editing Rules

- Keep the list curated. Do not add broad software categories unless they directly improve AI-assisted development.
- Prefer canonical GitHub repositories.
- Keep descriptions short, factual, and workflow-oriented.
- Do not add star counts or ranking language.
- Keep the README table of contents in sync when sections change.
- If a project has mixed licensing, write `License: see repository` instead of guessing.

## Curation Boundary

The core question for every entry is:

> Does this tool help developers or AI agents build, inspect, test, review, document, or ship software more effectively?

If the answer is only "it is useful software", it does not belong here.
